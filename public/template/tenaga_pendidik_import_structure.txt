STRUKTUR DATA IMPORT TENAGA PENDIDIK

Kolom yang WAJIB diisi:
- nama: Nama lengkap tenaga pendidik
- email: Alamat email unik
- tempat_lahir: Tempat lahir
- tanggal_lahir: Tanggal lahir (format: YYYY-MM-DD)
- no_hp: Nomor handphone
- kartanu: Nomor kartu NU
- nip: NIP Ma'arif
- nuptk: NUPTK
- npk: NPK
- madrasah_id: ID madrasah (lihat referensi di bawah)
- pendidikan_terakhir: Pendidikan terakhir
- tahun_lulus: Tahun lulus
- program_studi: Program studi
- status_kepegawaian_id: ID status kepegawaian (lihat referensi di bawah)
- tmt: Tanggal mulai tugas (format: YYYY-MM-DD)
- ketugasan: 'tenaga pendidik' atau 'kepala madrasah/sekolah'
- mengajar: Mata pelajaran yang diajarkan
- alamat: Alamat lengkap

Kolom yang OPSIONAL (boleh kosong):
- pemenuhan_beban_kerja_lain: 1 jika ya, 0 jika tidak (kosongkan jika tidak ada)
- madrasah_id_tambahan: ID madrasah tambahan (kosongkan jika tidak ada)

REFERENSI ID MADRA SAH (Contoh):
10 - SMA Ma'arif 1 Sleman
16 - SMK Ma'arif 1 Sleman
23 - SMK Ma'arif 1 Yogyakarta
...dan lainnya (lihat data madrasah lengkap)

REFERENSI ID STATUS KEPEGAWAIAN:
1 - PNS Sertifikasi
3 - GTY Sertifikasi
5 - GTY Non Sertifikasi
6 - GTT
...dan lainnya (lihat data status kepegawaian lengkap)

CATATAN PENTING:
- Email harus unik dan belum terdaftar
- Password akan dibuat otomatis menggunakan NIP (jika ada) atau default 'nuist123'
- Format tanggal: YYYY-MM-DD
- Ketugasan hanya menerima: 'tenaga pendidik' atau 'kepala madrasah/sekolah'
- File Excel HARUS memiliki baris header dengan nama kolom yang sesuai
