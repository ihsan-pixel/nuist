Susunan Data Import Tenaga Pendidik

Kolom-kolom yang harus ada dalam file Excel (.xlsx) untuk import tenaga pendidik, dengan header kolom sebagai berikut:

1. nama (Nama Lengkap Beserta Gelar) - Wajib
2. email (Email) - Wajib,
3. tempat_lahir (Tempat Lahir) - wajib
4. tanggal_lahir (Tanggal Lahir, format: YYYY-MM-DD) - wajib
5. no_hp (Nomor HP) - wajib
6. kartanu (Kartu NU) - wajib
7. nip (NIP) - wajib
8. nuptk (NUPTK) - wajib
9. npk (NPK) - wajib
10. madrasah_id (ID Madrasah) - wajib, harus sesuai dengan ID di database
11. pendidikan_terakhir (Pendidikan Terakhir) - wajib
12. tahun_lulus (Tahun Lulus) - wajib
13. program_studi (Program Studi) - wajib
14. status_kepegawaian_id (ID Status Kepegawaian) - wajib, harus sesuai dengan ID di database
15. tmt (TMT) - wajib
16. ketugasan (Ketugasan) - wajib
17. alamat (Alamat) - wajib

Catatan:
- File Excel HARUS memiliki baris header dengan nama kolom yang sesuai seperti di atas.
- Password akan dibuat secara otomatis (random 10 karakter).
- Role akan diset ke 'tenaga_pendidik'.
- Avatar/Foto Profile tidak diisi melalui import.
- Pastikan email aktif dan tidak ada duplikasi.
- Untuk madrasah_id dan status_kepegawaian_id, gunakan ID numerik dari database.
