PANDUAN IMPORT DATA TENAGA PENDIDIK

1. PERSIAPAN FILE:
   - Download template CSV dari sistem
   - Buka file dengan Microsoft Excel atau Google Sheets
   - JANGAN ubah baris header (baris pertama)
   - Isi data mulai dari baris kedua

2. PENGISIAN DATA:
   - Pastikan semua kolom WAJIB terisi
   - Email harus unik dan valid
   - Format tanggal: YYYY-MM-DD (contoh: 1990-01-01)
   - Ketugasan: pilih 'tenaga pendidik' atau 'kepala madrasah/sekolah'
   - ID madrasah dan status kepegawaian harus sesuai referensi

3. KOLOM OPSIONAL:
   - pemenuhan_beban_kerja_lain: isi 1 jika ya, kosongkan jika tidak
   - madrasah_id_tambahan: isi ID madrasah jika ada beban kerja di madrasah lain

4. VALIDASI DATA:
   - Periksa kembali data sebelum import
   - Pastikan tidak ada email duplikat
   - ID madrasah dan status kepegawaian harus ada di sistem

5. PROSES IMPORT:
   - Upload file Excel/CSV melalui menu Import
   - Sistem akan memvalidasi data otomatis
   - Jika ada error, perbaiki dan upload ulang
   - Password akan dibuat otomatis

6. SETELAH IMPORT:
   - Data akan tersimpan sebagai user dengan role 'tenaga_pendidik'
   - Password default: NIP atau 'nuist123'
   - Sarankan user mengubah password setelah login pertama

7. TROUBLESHOOTING:
   - Jika import gagal, periksa pesan error
   - Pastikan format file benar (.xlsx, .xls, .csv)
   - Periksa referensi ID yang digunakan

8. KONTAK SUPPORT:
   - Jika ada kendala, hubungi administrator sistem
