PANDUAN IMPORT DATA TENAGA PENDIDIK
==================================

Setelah migrasi database terbaru, ada perubahan struktur data yang perlu diperhatikan:

PERUBAHAN PENTING:
==================
1. Kolom 'ketugasan' sekarang menggunakan ENUM dengan nilai:
   - 'tenaga pendidik' (untuk guru/staf pengajar biasa)
   - 'kepala madrasah/sekolah' (untuk kepala sekolah/madrasah)

2. Kolom 'mengajar' adalah field BARU dan WAJIB diisi

FIELD YANG WAJIB DIISI:
======================
- nama
- email
- tempat_lahir
- tanggal_lahir
- no_hp
- kartanu
- nip
- nuptk
- npk
- madrasah_id
- pendidikan_terakhir
- tahun_lulus
- program_studi
- status_kepegawaian_id
- tmt
- ketugasan (gunakan nilai enum yang benar)
- mengajar (field baru, wajib diisi)
- alamat

CARA MENGGUNAKAN TEMPLATE:
==========================

1. TEMPLATE KOSONG (tenaga_pendidik_template.csv)
   - Gunakan sebagai dasar untuk membuat file import baru
   - Copy header dan isi dengan data Anda

2. CONTOH DATA TENAGA PENDIDIK (tenaga_pendidik_contoh.csv)
   - Berisi 10 contoh data tenaga pendidik biasa
   - Semua menggunakan ketugasan: 'tenaga pendidik'
   - Field mengajar diisi sesuai mata pelajaran

3. CONTOH DATA KEPALA MADRASAH (tenaga_pendidik_kepala_madrasah.csv)
   - Berisi 5 contoh data kepala madrasah/sekolah
   - Semua menggunakan ketugasan: 'kepala madrasah/sekolah'
   - Field mengajar diisi: 'Kepala Sekolah'

VALIDASI DATA:
==============
Sebelum import, pastikan:

1. Email harus unik dan belum terdaftar di sistem
2. Madrasah_id harus ada di tabel madrasahs
3. Status_kepegawaian_id harus ada di tabel status_kepegawaians
4. Ketugasan hanya boleh: 'tenaga pendidik' atau 'kepala madrasah/sekolah'
5. Field mengajar harus diisi (tidak boleh kosong)
6. Format tanggal: YYYY-MM-DD (contoh: 1990-01-15)
7. NIP harus berupa angka (tanpa spasi atau tanda hubung)

ERROR YANG MUNGKIN TERJADI:
===========================
1. "Field mengajar is required but empty" - Field mengajar kosong
2. "Invalid ketugasan value: Guru Matematika" - Menggunakan nilai lama
3. "Invalid email format" - Format email salah
4. "Madrasah ID X not found" - Madrasah_id tidak ada di database
5. "Status Kepegawaian ID X not found" - Status_kepegawaian_id tidak ada

CARA MENDAPATKAN ID REFERENSI:
==============================
1. Madrasah ID: Lihat di menu Master Data > Madrasah
2. Status Kepegawaian ID: Lihat di menu Master Data > Status Kepegawaian

CONTOH PENGGUNAAN:
==================
Untuk import data guru matematika:
- ketugasan: 'tenaga pendidik'
- mengajar: 'Guru Matematika'

Untuk import data kepala sekolah:
- ketugasan: 'kepala madrasah/sekolah'
- mengajar: 'Kepala Sekolah'
